# Alibaba Nacos配置中心demo
## Nacos Server
下载地址：https://github.com/alibaba/nacos/releases  
执行`./startup.cmd`启动  
访问：http://localhost:8848/nacos/  
账号密码：`nacos/nacos` 登录后台
